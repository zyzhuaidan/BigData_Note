## docker

### docker安装

- 环境

硬件: aarch64

系统: centos7.9

- 安装

 ```shell
yum -y install docker
```

- 安装成功

![img.png](img.png)

- 查看version

```shell
docker -v
```

- 开启docker为服务

```shell
systemctl start docker
```

- 设置docker开启启动服务

```shell
systemctlenabledocker
```
![img_1.png](img_1.png)

- 试试hello-world

```shell
docker run hello-world

docker images
```

![img_2.png](img_2.png)

- 配置阿里云镜像加速器

先去阿里云申请一个账号，登录之后在工作台搜索【容器镜像服务】找到【容器镜像服务】

```shell

sudo mkdir -p /etc/docker
sudo tee /etc/docker/daemon.json <<-'EOF'
{
  "registry-mirrors": ["https://你的地址.mirror.aliyuncs.com"]
}
EOF
sudo systemctl daemon-reload
sudo systemctl restart docker

```

### docker 进行常用命令

- 1. save
    
```
docker save tomcat:9.0.20-jre8 -o data/tomcat.tar
```
![img_5.png](img_5.png)


- 2. load
    
tar 转成镜像

```shell
docker load -i tomcat.tar
```

- 3. search
    
不建议使用， 不够直观

```shell
docker search tomcat
```

- 4. inspect
    
```shell
docker inspect tomcat:9.0.20-jre8
```

- 4. history

通过 docker history命令，可以列出各个层的创建信息
```shell
docker history tomcat:9.0.20-jre8
```
![img_6.png](img_6.png)

- 5. tag

标记本地镜像，将其归入某一仓库.

```shell
docker tag tomcat:9.0.20-jre8 tomcat:xingzhou
```

- 6. rmi
    
```shell
docker rmi tomcat:xingzhou
```

![img_7.png](img_7.png)

我们在使用 Docker 一段时间后，系统一般都会残存一些临时的、没有被使用的镜像文件，可以通过以下命令进行清理。
执行完命令后，还是告诉我们释放了多少存储空间！

```shell
docker image prune
```
